# ⏳ Backdrop Timer

A modern and minimal **Hackathon / Event Countdown Timer** built with **Flutter**.

Backdrop Timer is designed for hackathons, coding competitions, university events, and productivity sessions — providing a clean, reliable, and visually appealing countdown experience.

---

## 🚀 Overview

Backdrop Timer helps participants stay aware of remaining time during events.  
Built with Flutter, the app delivers smooth performance across Android, iOS, Web, and Desktop.

Whether it’s a 48-hour hackathon or a 2-hour sprint, this app keeps everyone aligned.

---

## ✨ Features

- ⏳ Custom Countdown Duration  
- ⏸ Pause & Resume  
- 🔄 Reset Timer  
- 🎯 Milestone Alerts (e.g., 12h left, 1h left)  
- 🌙 Dark Mode Support  
- 📱 Responsive UI  
- ⚡ Smooth Animations  
- 🔔 Local Notifications (optional)
